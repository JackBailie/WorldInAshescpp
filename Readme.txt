--Editor--
---Controls:---
1 - select tile placer tool
   lmb - place/select tile

2 - select collision box placer tool
   lmb(hold) - place collision box
   rmb - remove collision box

mmb - move camera

---files---
when the save button is pressed it will save a collision boxes file and a tile map file
these files should be placed in the game folder to import them into the game.

--Game--
---Controls---
WASD - move
